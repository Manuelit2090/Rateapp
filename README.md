# Rateapp
Proyecto de app web de reseñas con geolocalización, recomendaciones y un sistema de gamificación.

## Descripción

Es una plataforma web innovadora que optimiza la conexión entre la oferta gastronómica local y el consumidor final mediante geolocalización e inteligencia de preferencias. El sistema utiliza dinámicas de **gamificación** para incentivar la participación activa de los usuarios, transformando sus reseñas y visitas en beneficios tangibles como cupones de descuento. Simultáneamente, ofrece a los restaurantes un **panel de gestión estratégico** para exhibir sus productos, medir su impacto en el mercado y lanzar promociones dinámicas que atraigan tráfico en tiempo real
